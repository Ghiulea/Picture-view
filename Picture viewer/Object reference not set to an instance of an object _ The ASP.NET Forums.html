    Hi, move these lines

    If e.Item.ItemType = ListItemType.Item or e.Item.ItemType = ListItemType.AlternatingItem Then
          Dim btn as Button = CType(e.Item.Cells(3).FindControl("Delete"), Button)
          btn.Attributes.Add("onclick", "return confirm("blah blah")")    <-----------
    End If

    to the ItemCreated event of your DataGrid. Then, change this line

    <asp:ButtonColumn Text="Delete" ButtonType="PushButton" HeaderText="Delete" FooterText="Delete" CommandName="Delete">

    to

    <asp:TemplateColumn HeaderText="Delete">
       <ItemTemplate>
          <asp:Button id="Delete" CommandName="Delete" Text="Delete" Runat="server"/>
       </ItemTemplate>
    </asp:TemplateColumn>

    Pressing OK on the javascript confirm box will automatically cause post back and trigger the Datagrid ItemCommand event (assuming that you have it properly wired).    You can then use that event to call the delete method. Something like

    <code>
          Private Sub _attachmentGrid_ItemCommand(ByVal sender As System.Object, _
                                                    ByVal e As System.Web.UI.WebControls.DataGridCommandEventArgs) _
                                                    Handles _attachmentGrid.ItemCommand
               
        If e.Item.ItemType = ListItemType.Item OrElse e.Item.ItemType = _
                                                             ListItemType.AlternatingItem Then

      Dim attachmentID As String
             attachmentID = e.Item.Cells(0).Text

             Select e.CommandName

                Case "Edit"

       'do something

                Case "Delete"

                   Delete(attachmentID)

             End Select

        End If

       End Sub
    </code>

    HTH
    Tarun
